# FIFA World Cup 2026 Simulator 🏆

FIFAワールドカップ2026（アメリカ・メキシコ・カナダ共同開催）のグループステージおよび決勝トーナメント（ノックアウトステージ）の対戦結果を予測・シミュレートできるWebアプリケーションです。

## 🌟 主な機能

- **グループステージシミュレーション**: 全12グループ（A〜L）、48チームの対戦カードを網羅。各試合のスコアを自由に入力可能。
- **リアルタイム順位計算**: 各グループ内の順位表（勝ち点、得失点差、総得点など）を即時に計算。
  - 各グループ1位・2位（計24チーム）の自動勝ち抜けに対応。
  - 3位チームのうち、成績上位8チームがRound of 32（決勝トーナメント）に進出する2026年大会の独自レギュレーションを再現。
- **トーナメント（決勝ブラケット）の自動更新**: グループステージの結果に基づいて、Round of 32の対戦カードが自動的に決定。
- **ノックアウトステージシミュレーション**: Round of 32から決勝（および3位決定戦）までのスコア・PK戦結果を入力し、トーナメントの勝ち上がりを追体験。
- **自動シミュレート機能（強さ加味）**: 各チームに設定されたレーティング（実力値）に基づき、グループステージから決勝までの全試合を一括、または個別ステージごとに自動シミュレート可能。
- **全リセット機能**: シミュレーション結果をワンクリックでクリアし、初期状態からやり直すことができます。

## 🛠️ 技術スタック

- **フロントエンド**: React 19
- **開発言語**: TypeScript
- **ビルドツール**: Vite
- **スタイリング**: Vanilla CSS

## 📁 ディレクトリ構造

```text
src/
├── components/          # UIコンポーネント（対戦フォーム、順位表、トーナメント表など）
├── utils/               # 順位計算ロジック、トーナメント勝ち上がりロジックなど
├── types.ts             # 型定義（Team, Match, Standings, KnockoutMatch など）
├── data.ts              # 初期チームデータ、グループ分け、試合シミュレーションロジック
├── official_schedule.ts # 2026年ワールドカップの公式対戦日程データ
├── App.tsx              # メインアプリケーションコンポーネント
├── main.tsx             # エントリーポイント
└── index.css            # アプリケーションスタイル
```

## 🚀 開発環境のセットアップ

プロジェクトをローカル環境で起動・ビルドする手順です。

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

起動後、ターミナルに表示されるローカルURL（例: `http://localhost:5173`）にブラウザでアクセスしてください。

### 3. プロダクションビルドの作成

```bash
npm run build
```

### 4. ビルド後のプレビュー

```bash
npm run preview
```

---
*本シミュレーターは、FIFA World Cup 2026の大会方式（48チーム制・12グループ・32チーム決勝トーナメント）に基づいて設計されています。*
